UTF-8 (UCS Transformation Format — 8-bit) is a variable-width encoding that can represent every character in the Unicode character set.UTF-8 has become the dominant character encoding for the World-Wide Web, accounting for more than half of all Web pages. The Internet Engineering Task Force (IETF) requires allInternet protocols to identify the encoding used for character data, and the supported character encodings must include UTF-8.The Internet Mail Consortium(IMC) recommends that all e-mail programs be able to display and create mail using UTF-8. UTF-8 is also increasingly being used as the default character encoding in operating systems, programming languages, APIs, and software applications.
